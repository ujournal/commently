# Flarum як API бекенд для UJournal

[Flarum](https://flarum.org/) - фреймворк для створення спільнот. Цей фреймворк пропонує готовий функціонал і API. Його
підключено, як залежність. Репозиторій містить надбудови над фреймворком. Можна сприймати цей репозиторій, як власну збірку
бекенду Flarum, з обраними та власними розширеннями. Фронтенд Flarum не використовується і може існувати лише для тестування
або швидкої зміни налаштувань. Мета - максимально використовувати Flarum API і готових розширень.
