# Flarum як API бекенд для UJournal

[Flarum](https://flarum.org/) - фреймворк для створення спільнот. Фреймворк пропонує готовий функціонал і API. Фреймворк
підключено, як залежність. Репозиторій містить надбудови над фреймворком. Можна сприймати цей репозиторій, як власну збірка
бекенду Flarum, з обраними розширеннями, з метою використання API. Фронтенд Flarum не використовується і може існувати лише
для тестування або швидкої зміни налаштувань.
